STEP 1: Setup Backend (5 minutes)
----------------------------------
1. Open Command Prompt
2. Run:
   cd D:\Projects\Claude_Image\backend
   pip install -r requirements.txt

3. Copy .env.example to .env
4. Edit .env and add your Replicate API token:
   REPLICATE_API_TOKEN=r8_your_token_here

5. Run backend:
   uvicorn main:app --reload --port 8000

‚úÖ Backend ready at http://localhost:8000


STEP 2: Setup Frontend (10 minutes)
------------------------------------
1. Open NEW Command Prompt
2. Run:
   cd D:\Projects\Claude_Image
   setup_frontend.bat

3. Wait for setup to complete (2-3 minutes)

4. Copy files to frontend/src/:
   - frontend_files/App.js ‚Üí frontend/src/App.js
   - frontend_files/App.css ‚Üí frontend/src/App.css

5. Copy frontend_files/.env.example to frontend/.env

6. Run frontend:
   cd frontend
   npm start

‚úÖ Frontend ready at http://localhost:3000


STEP 3: Test the App
---------------------
1. Open http://localhost:3000
2. Enter business name: "My Business"
3. Select festival: Diwali
4. Click "Generate Image"
5. Wait 10-30 seconds
6. Download your image!

‚úÖ SUCCESS! Your AI Image Generator is working!
